# 💡 Li-Fi Based Data Transmission System

## 📌 Project Overview

This project demonstrates a Li-Fi (Light Fidelity) based data transmission system that uses visible light instead of radio waves for wireless communication. The system transmits data through an LED light source and receives it using a photodiode or solar panel.

## 🎯 Objectives

- Demonstrate wireless communication using visible light.
- Transmit data securely using LED light.
- Explore the applications of Li-Fi technology.
- Understand the principles of optical communication.

## 🛠️ Technologies and Components

- LED
- Photodiode / Solar Panel
- Audio Amplifier
- Resistors
- Capacitors
- Breadboard
- Connecting Wires
- Battery / DC Power Supply

## ⚙️ Working Principle

The transmitter converts electrical signals into light using an LED. The receiver detects the light using a photodiode or solar panel and converts it back into electrical signals. This enables wireless data transmission through visible light.
